# Plate Recognition Raspberry Pi

## Used devices

* Raspberry Pi Zero W
* Grove - Base Hat for Raspberry Pi Zero
* Grove - LCD Display 2x16 I2C v2.0 red-black
* Grove - Push Button with yellow LED
